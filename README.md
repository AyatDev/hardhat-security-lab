# Hardhat Security Lab

A ready-to-use Hardhat environment for smart contract security research and testing.

## Features

- Pre-configured Hardhat setup
- Slither integration
- Gas reporter
- Coverage reports
- Attack simulation scripts

## Setup

    npm install
    npx hardhat compile
    npx hardhat test
    npx hardhat coverage

## Structure

    contracts/
    ├── vulnerable/
    └── secure/
    test/
    scripts/
    └── simulate-attack.js
